# Background Images dos Cards

## Dimensões Recomendadas

**Tamanho ideal para cada imagem de background:**
- **Largura**: 400px
- **Altura**: 280px
- **Formato**: JPG (melhor performance) ou PNG (se precisar transparência)
- **Peso**: Máximo 150KB por imagem

## Arquivos Necessários

Crie 3 imagens com os seguintes nomes:

1. **bg-gerenciador.jpg** - Background do card "Gerenciador de Campanhas"
2. **bg-gerador.jpg** - Background do card "Gerador de Orçamentos"  
3. **bg-novo-modulo.jpg** - Background do card "Novo módulo"

## Onde Colocar as Imagens

Salve as 3 imagens nesta pasta:
```
workspace/assets/images/
```

## Como Subir para o GitHub

Após criar e salvar as imagens aqui, execute:

```bash
git add workspace/assets/images/
git commit -m "Add: Background images for workspace cards"
git push
```

O Render.com irá fazer deploy automático em 2-3 minutos.

## Dicas de Design

- Use imagens com **fundo claro** ou **gradientes suaves**
- Evite muitos detalhes (o texto precisa ficar legível)
- Cores sugeridas: tons de azul, roxo, verde ou gradientes
- O sistema já aplica um overlay branco semi-transparente (95% opacidade) sobre a imagem
- Você pode usar ferramentas como Canva, Figma ou Photoshop
- Sugestão: usar padrões geométricos, ilustrações minimalistas ou gradientes

**Card Gerador**: Gradiente roxo (#a855f7 → #ec4899)
